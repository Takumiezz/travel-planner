# 旅程 · 智能出行规划

> 一款以地图为核心的智能出行规划工具，融合交通路线规划与城市旅游攻略。

🌐 **在线体验：** [https://takumiezz.github.io/travel-planner](https://takumiezz.github.io/travel-planner)

---

## 📋 功能概览

### 🗺️ 地图主界面
- 基于 **Leaflet + 高德地图瓦片** 的全屏地图交互
- 所有地名、路名显示中文（国内 CDN 加速，8ms 延迟）
- 20 个城市以可点击标签形式展示在地图上
- 路线绘制动画、多方案对比

### 🔍 路线规划
- 输入出发地和目的地，自动生成出行方案
- 支持 **直达 + 1次中转 + 2次中转** 路线
- 四种排序策略：**推荐 / 最省时 / 最经济 / 最舒适**
- 地图上不同颜色区分不同方案
- 点击卡片查看行程时间线详情

### 📖 旅游攻略
- 每个目的地城市自动弹出右侧攻略面板
- 包含内容：
  - 🏛️ 必去景点列表
  - 🍜 必吃美食（含具体餐厅推荐）
  - 🏨 推荐住宿区域
  - 🗓️ 3 天详细行程路线
  - 📌 需提前预约的景点
  - 📸 拍照小贴士（机位、时间、穿搭）
  - 💡 实用旅行经验
  - 🚗 周边推荐
  - 💰 人均预算参考
  - 🚇 市内交通建议

### 🎛️ 筛选与排序
- 交通方式偏好（全部 / 仅高铁 / 仅飞机）
- 价格上限、耗时上限滑动条
- 最多中转次数限制
- 日期选择器

## 🚀 本地使用

直接打开 `travel-planner.html`（或 `index.html`）即可使用，无需安装任何依赖。

```bash
# 如果需要本地 HTTP 服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```
